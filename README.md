# Linear-Algebra-Hub
This is a website for linear algebra knowledge implemented with AI, based on Flask, Python, and MySQL. You can search for the knowledge you want on it, and you can also ask questions and leave messages anytime and anywhere, answer others' doubts, and solve your own doubts. The website aims to introduce linear algebra knowledge, and its functions include user registration and login, knowledge point query, discussion area, search, private message, collection, notification, comment and rating, etc. However, due to limited time, only 100 linear algebra knowledge points have been added to the database, and it is not yet complete. If there is time in the future, the website will be considered to be improved and released.
